# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-01

### Added
- Initial release
- AI code generation using Kimi API
- Live preview with Sandpack
- Export as ZIP (complete Next.js project)
- Penpot SVG bridge for design integration
- Penpot self-hosting with Docker Compose
- Vagrant VM deployment option
- Comprehensive documentation

### Features
- Next.js 16 + React 18 + TypeScript
- Tailwind CSS + shadcn/ui components
- Dark mode UI
- Copy to clipboard
- Quick example prompts
- Responsive design

### Deployment Options
- Docker Compose (full stack)
- Vagrant + VirtualBox
- Native development

### Documentation
- README with badges
- Architecture documentation
- API reference
- Deployment guide
- Vagrant setup guide

- Contributing guidelines
